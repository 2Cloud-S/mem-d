# Mem-D Website Specification (V1)

## Design Goal

The website should feel like:

* A developer tool
* A research product
* An observability utility

The website should NOT feel like:

* An AI startup
* A chatbot company
* A SaaS landing page template

---

# Visual Language

## Tone

Practical

Technical

## Avoid

* Purple gradients
* Cyan glow effects
* Neon accents
* Glassmorphism
* Floating cards
* Abstract AI artwork
* Robot illustrations
* Fake dashboards
