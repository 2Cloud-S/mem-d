## 3. Pipeline Integration

`analyze_file()` chains immutable updates:

```python
report = report.model_copy(update={"simulationReport": simulate_recommendations(report)})
return report.model_copy(update={"workflowPlan": plan_workflows(report)})
```

- `plan_workflows()` invoked exactly once, after simulation
- Default `PlanningOptions(includeKeep=False)` via planner default
- `apply_workflow_decisions()` not called
- Planner exceptions propagate; integrity-blocked plans attach as data

---

## 4. JSON Serialization

Single boundary helper:

```python
def workflow_plan_to_dict(plan: WorkflowPlan) -> dict[str, object]:
    return plan.model_dump(mode="json", exclude_none=False)
```

`report_to_dict()` omits `workflowPlan` when `None`; includes complete nested plan when present. No child serializers; no `plannerSignalInputs`.

---

## 5. Markdown Rendering

- `render_workflow_plan_markdown()` added; returns `[]` when plan absent
- Inserted after Simulation, before Compression Explanation
- Structured fields only; operation data from `WorkflowStep.operation`
- Deterministic truncation limits (steps/items 20, overridable blockers 20, evidence refs 20 per category; integrity refs never truncated)
- Planning-only disclaimer on every workflow section

---

## 6. Terminal Rendering

- `render_workflow_terminal()` added; called after simulation terminal block
- Shows aggregate status, planner state (`initial`), counts, structural delta, top 3 queues, up to 3 blockers (integrity first)
- Omits ready-for-execution count
- Notice: `Planning only — operator decisions have not been applied.`

## 7. Heading Hierarchy Correction

Simulation summary demoted for valid peer structure:

| Before | After |
| --- | --- |
| `# Simulation Summary` | `## Simulation Summary` |
| `## Projected Store Impact` | `### Projected Store Impact` |
| `## Recommendation Outcomes` | `### Recommendation Outcomes` |
| `## Structural Estimates` | `### Structural Estimates` |
| `## Warnings` | `### Warnings` |

`## Workflow Plan` is a peer of `## Simulation Summary`, not a subsection.
