## Known Limitations

1. **Not wired into `analyze_file`** — recommendations are callable via `plan_recommendations()` only.
2. **No report/CLI output** — JSON, Markdown, and terminal renderers unchanged.
3. **`AnalysisReport` unchanged** — `recommendations` and `memoryResolutions` fields deferred to Phase 2.
4. **`include_keep` defaults false** — keep recommendation records omitted unless flag set; per-memory resolution still defaults to `keep` for Active memories without competing signals.
5. **Gold fixture uses embedded validation** — not all cases run full pipeline; lifecycle integration test covers one end-to-end evolution → lifecycle → recommendation path.
6. **No governance recommendation accuracy benchmark** — gold fixture provides case-level checks only; no precision/recall rollup yet.
7. **`defer` recommendations** — only emitted when `include_deferred=True`.
