## Remaining limitations

1. Gold fixture has 9 synthetic cases — architecture-compliance benchmark, not production-world coverage.
2. LongMemEval/PERMA regression guards not automated in CI at launch.
3. `simulationMode` variants (`merge_only`, `archive_only`) not evaluated in Phase 3.
4. Pre-existing parser test failure (`memories.txt` missing) unrelated to Phase 3.
